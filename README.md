# Lead Management System

## Overview

A full-stack CRM application developed using React, Node.js, Express.js, and MongoDB.

## Features

* Secure Admin Login
* Add New Leads
* View Lead Listings
* Update Lead Status
* Delete Leads
* Follow-up Notes
* Dashboard Analytics

## Tech Stack

### Frontend

* React
* Vite
* Axios
* CSS

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication
* bcryptjs

## Project Structure

* frontend/
* backend/
* outputs/

## How to Run

### Backend

npm install

npm run dev

### Frontend

npm install

npm run dev
